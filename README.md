# Ropasc CE #

Ropasc CE is a digital adaptation of a traditional game called "rock paper scissors" for handheld PCs where the human player competes against the CPU. It was written in C and uses WinAPI.

### How to play? ###

You play against the computer. When you perform a move then the computer's move is revealed immediately. You can use one of the three items: rock, paper or scissors. Rock beats scissors, paper beats rock and scissors beats paper. If both the player and the computer use the same item, the round ends with a draw.

### Game options ###

Two UI languages are available: English (default) and German. Restart is not required but if the language is other than English then you will have to select it each time you launch the application.

### Recommended programming setup ###

Source OS: Windows

Target OS: Windows CE (min. 1.0)

IDE: Visual C++ Professional Edition, version 5.0

Visual C++ for Windows CE
